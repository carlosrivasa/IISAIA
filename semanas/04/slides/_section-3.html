Leer vs. escribir — el riesgo no es simétrico

Leer

reversible

Leer un archivo, listar un directorio, llamar un endpoint de solo lectura. El entorno no cambió. El peor caso: llenó el contexto con información irrelevante.

Puede intentar otra lectura.

Escribir / ejecutar

puede no ser reversible

Editar un archivo, correr git commit, borrar un directorio, llamar un endpoint que muta estado. El entorno quedó en un estado diferente al esperado.

No hay deshacer automático.
