Por qué conecta con todo lo del curso

S2 / S3
vocabulario
Componentes, endpoints, contratos, status codes — lo que te permite dictar. Si el agente devuelve una API que no pediste, necesitás las palabras para corregirla. Sin vocabulario, el único movimiento es aceptar o rechazar en bloque.
S4
context y harness
Entran cuando el proyecto crece. CLAUDE.md persiste las decisiones de arquitectura entre sesiones. Los sub-agents aíslan partes del loop. Plan mode muestra el plan antes de tocar nada. En un proyecto real de varios archivos, dejan de ser opcionales.
S3
supervisor
El rol de supervisor mantiene la coherencia entre partes. Si el frontend espera un campo y el backend devuelve otro, el agente no lo detecta solo. Supervisar el contrato entre cliente y servidor es exactamente el rol de la semana 3.
El trabajo final no examina si sabés sintaxis. Examina si sabés dirigir.

Qué se evalúa — en espíritu

No es solo que funcione. Un producto que funciona pero cuyos caminos no podés explicar no dice nada sobre tu rol como supervisor. El término para esto es apropiación: que el producto sea tuyo aunque el agente haya escrito la mayoría del código.
explicar
por qué se tomaron las decisiones de arquitectura que se tomaron
corregir
cuando el agente derivó o cuando el contrato entre partes se rompió
extender
agregar algo al proyecto sin empezar de cero, gestionando el contexto acumulado
Apropiación no se mide en cuántas líneas escribiste con el teclado.

Un producto que funciona pero cuyos caminos no podés explicar no demuestra nada sobre tu rol como supervisor — demuestra que el agente funcionó, no que vos lo dirigiste."

"El término para lo que sí importa es apropiación. Que el producto sea tuyo aunque el agente haya escrito la mayoría del código. Apropiación se mide en tres verbos."

"Explicar: por qué estos endpoints, por qué esta estructura de datos, por qué este contrato entre cliente y servidor. Esas decisiones tienen que ser tuyas — no las que el agente eligió por defecto porque no especificaste."

"Corregir: cuando el agente derivó, cuando el contexto se fragmentó entre sesiones, cuando el frontend y el backend dejaron de hablar el mismo idioma — que puedas detectarlo y corregirlo."

"Extender: agregarle algo al proyecto sin empezar de cero. Gestionar el contexto acumulado de semanas anteriores para que el agente entienda la arquitectura antes de agregar la próxima feature."

"Eso es apropiación. No cuántas líneas escribiste."
