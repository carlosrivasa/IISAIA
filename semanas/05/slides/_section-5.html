Comando único contra el marketplace oficial.

"Un solo comando. Como el marketplace oficial viene pre-registrado, no hace falta agregar nada antes — abrís Claude Code en una terminal limpia y corrés esta línea."

"Esto baja el plugin a ~/.claude/plugins/cache/claude-plugins-official/superpowers/ y registra todas sus skills. Cuando termine, el flujo que vimos en el roadmap queda disponible de una. No hay que reiniciar Claude Code."

Verificar que cargó

$ /plugin list

Dos planes B documentados: marketplace alternativo y reload-plugins.

"Dos contingencias documentadas. La primera, si el marketplace oficial está caído o el comando timeoutea: existe un marketplace comunitario alternativo de la comunidad, mantenido por el autor original del plugin. Lo nombramos, no lo corremos en vivo."

"La segunda, si el install termina sin errores pero las skills no figuran en /plugin list: ejecutás /reload-plugins. Es el procedimiento documentado en la doc oficial para cuando el registro de skills queda colgado entre sesiones."
