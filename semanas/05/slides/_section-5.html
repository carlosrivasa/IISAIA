Instalá Superpowers — un solo comando

Las instrucciones, en una pantalla

1. instalar
/plugin install superpowers@claude-plugins-official

El oficial viene pre-registrado. No hace falta marketplace add antes.

2. verificar
/plugin list
$ /plugin list

superpowers  v5.1.0  claude-plugins-official  enabled

Si algo no coopera

El marketplace oficial no responde. Usá el comunitario:

/plugin marketplace add obra/superpowers-marketplace
/plugin install superpowers@superpowers-marketplace

El install terminó pero las skills no figuran en /plugin list:

/reload-plugins
